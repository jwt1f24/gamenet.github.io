Trading Card Game by Indie Malaysian Developer has been Gaining Attention Lately

"#SupportLocal."

Article published by: @jessepinkman17

Credit: OPNeon Games

If you browse through Steam for new games to play, you might have noticed this one game about card trading. This game has started to gain a lot of traction lately, but how did it became this popular? Well, we would have to go all the way back to a post on Reddit.

In the r/Bolehland subreddit, a user by the alias, OPNeon, which is the sole developer of the TCG Simulator game, made a post promoting his game whilst revealing that he is a local from Malaysia.

This definitely shocked fellow Malaysians, as they did not expect that such a popular game was made by a Malaysian. Afterwards, more Malaysians started giving the developer's game a try and many liked the game.

The game continued to grow in popularity, to the point where even famous streamers were playing it live as well. Have you tried it? If you haven't, why not support our local growing game developers?
